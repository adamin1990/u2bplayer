# u2bplayer
youtube  播放器 

##技术点

* auth认证登录
* xml解析
* json解析 
* google admob
* dragpannel （youtube播放页效果）
* 搜索
* 数据库操作
* asytask 异步更新UI
* PagerSlidingTabStrip
* 视频播放 
* youtube sdk的使用等等 .
